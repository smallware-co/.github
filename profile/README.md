# Smallware Co.

Small tools. Real utility. Bring your own AI key.

All tools are free to use, open source, and run entirely in your browser. No backend. No data collection. Your API key never leaves your device.

### Content Tools

| Tool | What it does |
|------|-------------|
| [content-brief-builder](https://smallware-co.github.io/content-brief-builder) | Keyword in. Full brief out — audience, intent, H2 structure, title options, and word count. |

## // byok — bring your own key

Most tools support Gemini, OpenAI, and Anthropic keys. A few use Z.AI GLM. Keys are stored in your browser's local storage only and are never sent to any server we control. Check the source on any repo to verify.

Two tools (Skill Finder and PowerPlanner) need no key at all.
