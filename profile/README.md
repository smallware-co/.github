# Smallware Co.

We build small, useful software for everyday problems.

## Apps

### [PowerPlanner](https://github.com/smallware-co/powerplanner) *(in testing)*
An off-grid power calculator. Size your solar, wind, hydro, generator, and battery setup for life off the grid.

### [Fuel Tracker](https://github.com/smallware-co/fuel-tracker) *(in development)*
Track fuel consumption, calculate KM/L, and log fill-ups. PWA-first with OCR receipt scanning.
